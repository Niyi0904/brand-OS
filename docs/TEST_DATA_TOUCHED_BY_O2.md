## Adding Groq for Real AI

1. Get key: https://console.groq.com/keys
2. Add to `.env`:
   ```env
   GROQ_API_KEY=gsk_your_key_here
   ```
3. Restart: `npm run dev`
4. Test again - should get real AI responses with Brand Brain context

## Troubleshooting

**No employees showing on roster:**
- Run: `npm run db:seed`
- Hard refresh browser (Ctrl+Shift+R)
- Check database connection in `.env`

**Chat not working:**
- Check browser console (F12) for errors
- Verify you're logged in
- Try direct URL: `/dashboard/employees/marketing-director`

**No AI responses:**
- Add Groq API key to `.env`
- Or use mock responses (no key needed for UI testing)
